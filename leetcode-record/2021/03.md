| 575 | easy  | Distribute Candies |

```java
class Solution {
    public int distributeCandies(int[] candyType) {
        int len = candyType.length;
        Arrays.sort(candyType);
        int count = 1;
        for (int i = 0, j = 0; i < len; i++) {
            if (candyType[i] != candyType[j]) {
                j = i;
                count++;
            }
        }
        return Math.min(count, len / 2);
    }
}
```
