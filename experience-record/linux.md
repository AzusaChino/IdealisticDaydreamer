# Linux相关经验

## Ubuntu更新APT源

```sh
cd /etc/apt/
cp sources.list  sources.list.bak
true > sources.list
vi  /etc/apt/sources.list    # 将上面的源复制到文件中, 保存退出
apt update    # 更新本地 apt 源
```
